#### Create new User

1. login to the remote system
```bash
ssh ubuntu@1.1.1.1
```

2. Create new user - `jupyteruser`
```bash
sudo adduser jupyteruser
```

3. Grant the user administrative right:
```bash
sudo usermod -aG sudo jupyteruser
```
